# Cloud-Native-SaaS-Application
AWS CLI - kubectl - Terraform - Docker - Git - Jenkins ( demonstrating CI/CD, Kubernetes,
Terraform, and monitoring on AWS.)

Architecture
- AWS EKS
- Docker
- Jenkins CI/CD
- Terraform (IaC)
- Prometheus & Grafana

Deployment Flow
1. Code push to GitHub
2. Jenkins builds & pushes Docker image
3. Kubernetes deploys updated application
4. Monitoring tracks performance
